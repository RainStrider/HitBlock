🎮 HitBlock - 子弹射击方块小游戏
基于 Cocos Creator 3.8.2 开发的轻量级休闲射击小游戏
项目简介
HitBlock 是一款操作简单、上手即玩的休闲射击小游戏。玩家通过发射子弹击碎场景中的方块，体验轻松爽快的消除射击乐趣，支持PC 端 + 移动端浏览器运行。
核心玩法
控制发射角度 / 自动发射子弹
击碎所有方块即可完成关卡
轻量化设计，无广告、无内购，随时随地畅玩
技术栈
游戏引擎：Cocos Creator 3.8.2
发布平台：Web Mobile（H5）
部署平台：GitHub Pages
