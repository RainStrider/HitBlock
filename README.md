# HitBlock - 子弹射击方块小游戏

一款基于 Cocos Creator 3.8.2 开发的轻量级休闲射击 H5 小游戏

## 🎯 游戏简介
HitBlock 是一款上手零门槛的射击消除小游戏，玩家通过发射子弹击碎场景中的方块，完成闯关挑战。
游戏支持移动端、PC 端浏览器直接运行，无需下载，点开即玩。

## 🕹️ 核心玩法
- 发射子弹击打场景中的方块
- 轻量化休闲玩法，适合碎片化时间体验
